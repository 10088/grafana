# Icon

Grafana's wrapper component over [Font Awesome](https://fontawesome.com/) and Unicons icons.


### Changing icon size

By default `Icon` has width and height of `16px` and font-size of `14px`. Pass `className` to control icon's size:

```jsx
<Icon name="check" />
```
